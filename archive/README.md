## To Complete a Skill

1. Create SKILL.md with YAML frontmatter and instructions
2. Create PROFILE.md.template
3. Write comprehensive tests
4. Update documentation
5. Move to `/new-skills/` (if Python-powered) or `/superskills/` (if prompt-only)
