## Tips for Best Results

### Be Specific Initially
The more detail you give the Manager upfront, the better the output:
- Content type and length
- Target audience
- Key frameworks to include
- Tone and style preferences

### Trust the Process
Each agent has specific expertise. The Manager knows when to use which agent and when to request revisions.

### Iterate Freely
Tell the Manager what needs adjustment:
- "Make it more conversational"
- "Add more examples"
- "Too technical for this audience"
- "This doesn't sound like me"

### Use the Templates
For complex projects, fill out `templates/project-brief-template.md` and give it to the Manager.
